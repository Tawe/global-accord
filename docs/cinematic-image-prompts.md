## Recommended Generation Specs

- Aspect ratio: `16:9`
- Resolution: `1920x1080` minimum
- Style: consistent painterly / polished cinematic illustration
- Avoid:
  - text in image
  - logos
  - UI elements
  - extreme saturation
  - comedic or overly heroic tone
